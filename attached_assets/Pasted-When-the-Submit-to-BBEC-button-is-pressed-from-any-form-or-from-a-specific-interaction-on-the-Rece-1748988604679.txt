When the "Submit to BBEC" button is pressed from any form or from a specific interaction on the Recent Interactions, or History sections:  Initiate an API call to BBEC using the DataFormSave method (details below) to insert the record.  Once you receive a 200 code signaling that the API call succeeded, the interaction record can be removed from this database and all lists.  For the API data mapping, <ContextRecordID> hould be filled with the constituent BBEC GUID fro the interaction form, <fv ID="STATUSCODE"> should always be 1 for now (this will change in the future), <fv ID="FUNDRAISERID"> should always be the User Profile GUID from the settings tab, <fv ID="INTERACTIONTYPECODEID"> should always be D081262B-4430-416D-8A6E-D0CBDFFDBC8F for now, <fv ID="EXPECTEDDATE"> should be the Actual Date of the interaction,  <fv ID="OBJECTIVE"> should be the value in the Summary field of the interaction, <fv ID="COMMENT"> should be the value in the Detailed Comments field of the interaction.  Here are the API details in an example that works in POSTMAN - replace the example values with the variables described above.
